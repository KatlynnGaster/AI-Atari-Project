Overview
This project implements an Approximate Q-Learning agent trained to play Atari Bowling. 
The network uses hand-engineered visual features pulled from the game frames. 
The goal of this project is to explore how effective reinforcement learning can be without using Deep Q-Learning. 

Environment
ALE/Bowling-v5 in Gymnasium framework

Training
During training, scores are printed per episode, successful runs are recorded automatically, and performance logs are saved. 

Results
The agent demonstrates gameplay strategies that are meaningful from the visual features, linear function approximation, and reinforcement learning.
